<h1>층간 소음 측정 및 관리 공유 커뮤니티 서비스</h1>


<h2>프로젝트 개요</h2>

  <ul>
    <li>
      최근 증가하는 세종시의 층간소음 민원에 대응하여 빅데이터, IoT, AI, 클라우드 융합기술을 활용한        데이터 기반 관리 서비스를 개발하여 층간소음 문제를 예방 및 해결
    </li>
    <li>
      세종시의 총 인구수 연평균 증가율은 5.12%, 연 평균 증가율은 21% 기록
    </li>
    <li>
      2022년 접수된 층간소음 민원의 71%가 아이들 뛰거나 걷는 소리
    </li>
  </ul>
